1.1 Color Model

https://www.nceas.ucsb.edu/~frazier/RSpatialGuides/colorPaletteCheatsheet.pdf

color wheel

R can use many different color models, but often uses a #Red, Green, Blue model with the amount of each color indicated in Hexadecimal, 00 to FF, notation.

1.2 Three Types Of Palettes

http://colorbrewer2.org/

library(RColorBrewer)
display.brewer.pal(7,"Set1") # qualitative

display.brewer.pal(7,"YlOrRd") # sequential

display.brewer.pal(7,"Spectral") # diverging

display.brewer.all() # all palettes
